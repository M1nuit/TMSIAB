*****v0.8.2 -> v0.9.0*****
**************************
- Neu: Dedimania vollkommen integriert
- Neu: Neue Ladderpoints werden am Rundenanfang im Chat angezeigt. Es ist allerdings auch möglich, dass Ladderpoints angezeigt werden, die zuvor auf einem anderen Server gefahren wurden.
       Diest ist leider nicht anders zu lösen.
- Neu: Pluginmanager - Ermöglicht jedem SuperAdmin das editieren der aktiven Plugins ingame! Plugins lassen sich einfach durch das Fenster, welches beim Command '/plugins' erscheint, aktivieren und deaktivieren.
                       Einstellungen (welche Plugins nicht verändert werden dürfen etc.) können in der Datei 'plugins/manager.plugins.php' vorgenommen werden.
- Neu: WindowClass   - Diese neue Klasse ermöglicht es Entwicklern, ganz einfach Fenster darzustellen, die dem Design von FoxControl entsprechen.
                       Mit der WindowsClass entstehen keine Überlappungen der Fenster, da sie automatisch geschlossen werden, wenn ein neues Fenster aufgemacht wird.
                       Auch kann der Entwickler Tabellen und Buttons mit geringem Aufwand hinzufügen
                       Eine Dokumentation wird bald folgen.
- Neu: Jukebox und ChallengeList sind nun getrennt -> Einfaches Deaktivieren von einem der beiden ist möglich. Eine grafische Darstellung der Jukebox kommt im nächsten Update.
- Neu: Das Willkommensfenster kann nun in der config.xml editiert werden.
- Bug Fix: Fehler im plugin.players.php - /afk Bug behoben
- Bug Fix: Fehler im plugin.menu.php - Kleinere Bugs behoben, sowie die Nicknames rausgenommen
