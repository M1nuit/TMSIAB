Welcome to Trackmania Server in a Box ( United/Nations Forever-Version )

The purpose of TMSiaB is a to euqip Trackmaniacs with a simple tool, that includes all to configure, start and administer a Dedicated Server. Just a few clicks and all relevant datafiles as may be strictly required will be configured automatically.

To configure your Dedicated Server you have to fill in the desired data in each Tab at New Server and finally "Create".

To start the whole shebang you have to click on three more buttons under Start Server: "Start Webserver", "Start Dedicated" and "Start Plugin" - that´s near idiotproof, isn't it? ;O)

Shortly after, everybody can join the created Dedicated Server and you´ll be able to login at remoteCP with your TM-Loginname and password "admin". Having done this you´ll be able to add Tracks, setting up "Matchsettings", restart rounds and all the rest of it.

Have fun!
